# NRfood

Сайт-визитка службы доставки халяль стритфуда «NRfood» (Кокшетау).
Крылышки, комбо-сеты, пицца, донер и гриль. Доставка горячим за 45 минут. RU/KZ.

Главные фишки: витрина **комбо-сетов**, табовое **демо-меню** (помечено как демонстрационное)
и заказ через WhatsApp.

## Стек
Next.js 14 (App Router) · TypeScript · Tailwind CSS · Framer Motion · next-intl (RU/KZ) ·
self-hosted шрифты (@fontsource: Anton, Oswald, Manrope) · lucide-react.

## Запуск
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # продакшн-сборка
```
